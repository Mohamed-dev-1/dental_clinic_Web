# Dental Clinic Web Application

## Overview:
this is a web application that serves Dental Healthcare Facilities, its helps the management of the various aspects of the facility, such as the **Appointements**, the **Accountes** of both the patients or the workers and many other features

## Project Structure:
the Project Folder Structure are as follows
`

        root-folder
            |
            |
            |------ resources/
            |           |
            |           |_________ (Photos, Multimedia files ...)
            |
            |------- styles/
            |           |
            |           |_________ (Stylesheet files)
            |
            |------- scripts/
                        |
                        |_________  (Javascirpt Script files)

`
